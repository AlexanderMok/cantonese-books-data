說明
----

以下列出「反切標記」欄可能出現的值及其意義。數字 1-5 是校訂者對勘誤切所作之附註，僅供參考。

```
N　讀音不屬反切
X　讀音不是正確的反切
E　反切在勘誤表判為誤讀
1　勘誤切讀音已見於釋義
2　勘誤切讀音可疑
3　原讀亦有理
4　原讀合今音、勘誤合廣韻反切
5　原讀合廣韻反切、勘誤合今音
```

說明
----
 * 原書以簡化字排印，本讀音資料檔只收繁體。
 * 若簡體另兼正體，視情況區分處理。如「蔔」簡化作「卜」，但「卜」另兼正體而本身沒有來自「蔔」的簡化字的baak6音。此情況會分為兩個獨立字條，其中「卜」只收buk1音、「蔔」只收baak6音。
 * 一簡對多繁而讀音完全相同者或會在同一字條列出。
 * 字目後除錄入繁體，亦會收入異體寫法。這類異體字本檔會酌收，惟不完整。
 * 非破讀字不收釋義。破讀字會酌收義項或例詞，旨在扼要區別字義，或經刪減，未必悉依原文。敬請參照原書是荷。
